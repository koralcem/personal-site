Burgle Bros Recon

Problem: Burgle Bros is a cooperative board game, where a team of thieves explore an office building in order to steal what's in the safe. You reveal room tiles as you go, but alarm tiles cause problems. The game provides a reference card of how many of each tile is in the game, so you can technically look at what's been explored and try to guess what the next tile can be. But counting all the explored tiles is tedious.

Solution: I digitized the reference card. You start out with all tiles in the game. Click on them to keep track of what you've explored, and see at a glance what's left. I made the same tool both with React, and with web components.

Results: Less time spent counting tiles, and more time enjoying your game and the company of your friends.

Phonetic Speller

Problem: My full last name is a mess (ask me about it). When I call customer support for anything, I'd like to use the phonetic alphabet to spell it out precisely, but I can never remember the code words.

Solution: I created a tool where you enter a word or phrase, and get the code words for each of its letters.

Results: No ambiguity over the phone when pulling up my account.

Psychological Services Institute website
